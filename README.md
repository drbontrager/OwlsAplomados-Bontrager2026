# OwlOccupancy

Nimble code used for the integrated occupancy model in Bontrager et al. (2025) 'Woody plant encroachment in grasslands benefits two predatory owl species, threatening the recovery of Aplomado Falcons'

The code uses MCMC in the Nimble package to estimate occupancy of Great Horned Owls and Barn Owls on Laguna Atascosa National Wildlife Refuge in coastal South Texas.
